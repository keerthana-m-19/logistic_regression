# Logistic Regression ML Project

## 📌 Overview
This project demonstrates the use of **Logistic Regression**, a supervised machine learning algorithm, for solving a **binary classification problem**.  
We predict whether a customer will purchase a product based on their **Age** and **Estimated Salary**.

---

## ⚙️ Steps in the Project
1. Import libraries and dataset
2. Data preprocessing (scaling features)
3. Train-test split
4. Model training with Logistic Regression
5. Model evaluation using:
   - Accuracy Score
   - Confusion Matrix
   - Classification Report
6. Visualization of results
7. User input for prediction (Age & Salary)

---

## 📊 Results
- Achieved good accuracy on the test data
- Visualized confusion matrix
- Interactive predictions for new customers

---
